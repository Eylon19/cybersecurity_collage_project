פרויקט: אתר להגברת מודעות לסייבר

טכנולוגיות:
NodeJS, Express, EJS, MySQL, mysql2
מבנה MVC: routes, controllers, models, views

הפרויקט כולל:
1. הרשמה והתחברות.
2. 4 טבלאות: users, articles, threats, quiz_results.
3. CRUD למאמרים.
4. CRUD לאיומי סייבר.
5. דפים דינמיים עם EJS.
6. אזור אישי שמציג פרטי משתמש ותוצאות מבחנים.
7. חלק פתוח וחלק חסום למשתמשים מחוברים.

הוראות הרצה:
1. לפתוח את התיקייה ב-Visual Studio Code.
2. להריץ בטרמינל:
   npm install
3. לפתוח MySQL Workbench ולהריץ את הקובץ db.sql.
4. לפתוח את util/database.js ולעדכן את הסיסמה של MySQL במקום YOUR_PASSWORD.
5. להריץ:
   node app.js
6. לפתוח בדפדפן Chrome:
   http://localhost:3000

הערה:
הפרויקט בנוי לפי החומר של הקורס: Express, MVC, Router, Controller, Model, EJS, MySQL, forms, post/get, res.render, res.redirect.
